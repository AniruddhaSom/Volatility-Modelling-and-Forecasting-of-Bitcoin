# Volatility Modeling and Forecasting of Bitcoin

## Project Overview
This project analyzes Bitcoin price data (2013–2017) to study price trends, volatility, and market behavior using time series techniques and machine learning.

## Objectives
- Analyze Bitcoin price movements over time
- Study volatility using rolling statistics
- Identify trends using moving averages
- Forecast future prices using Prophet model

## Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib, Seaborn
- Prophet (Machine Learning)
- Jupyter Notebook

## Key Analysis
- Data cleaning and preprocessing
- Time series visualization (Open, Close, High, Low)
- Rolling Mean and Rolling Standard Deviation (Volatility)
- Moving Averages (SMA, EMA)
- Candlestick chart analysis
- Time-based aggregation (daily, monthly, yearly)
- Forecasting using Prophet

##  Results & Insights
- Bitcoin shows high volatility with sharp price fluctuations
- Periods of high returns are associated with high risk
- Moving averages help identify trend shifts
- Prophet model captures overall trend but may not capture extreme spikes
